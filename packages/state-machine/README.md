# @spike-land-ai/state-machine

Statechart engine with guard expression parser for `spike.land`. This package provides a robust and flexible way to manage complex states and transitions in the `spike.land` platform.

## Features

- Flexible statechart engine with support for guards and actions.
- Integrated guard expression parser for advanced state transitions.
- Optimized for performance and readability.

## Installation

```bash
yarn install
```

## Build

```bash
yarn build
```

## Usage

### Development

```bash
yarn dev
```

### Run Tests

```bash
yarn test
```

### Typecheck

```bash
yarn typecheck
```
