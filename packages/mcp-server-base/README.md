# @spike-land-ai/mcp-server-base

Shared base utilities for `@spike-land-ai` MCP server packages. This package provides core functionality and helpers that are common across multiple Model Context Protocol servers.

## Features

- Shared base classes for MCP servers.
- Common utility functions and middleware.
- Consistent configuration and logging patterns.

## Installation

```bash
yarn install
```

## Build

```bash
yarn build
```

## Usage

### Development

```bash
yarn dev
```

### Run Tests

```bash
yarn test
```

### Linting

```bash
yarn lint
```
