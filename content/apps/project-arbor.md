# Project Arbor

Project Arbor packages one thesis into a usable MCP app:

1. create capital from local inputs
2. route labor into protected income
3. formalize and defend fragile gains
4. keep distribution alive when infrastructure is weak or hostile

The app is built for strategy work before field rollout. Use it to turn the concept into a concrete operating plan.
