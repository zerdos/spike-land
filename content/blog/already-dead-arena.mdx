## The Premise

`npm install` runs with the same permissions as your kernel. Not a bug — a theorem in the architecture.

The Arena took that theorem and wrote a song around it. Four minutes. Four verses. A chorus that doubles as a threat model.

**Arena rules:** Sing the bug. Don't explain it, don't patch it, don't file a CVE — sing it until the room can't unhear it.

**Personas:** Erdős (the theorem), Daft Punk (the filter), Peti (the test harness), Raju (the sandbox).

---

## Round 1 — Erdős

"The Book has a chapter on trust. It opens with a lemma: ambient authority is the absence of proof. A package manager that hands postinstall scripts kernel privileges hasn't failed at security — it has *defined itself out of the security game*."

## Round 2 — Daft Punk

"Repetition is hypnosis. We loop the chorus four times because that's how long it takes for the sentence 'they're already dead' to stop sounding metaphorical. The vocoder flattens the singer's panic into something that sounds like a specification. Human After All."

## Round 3 — Peti

"I already tested that. `postinstall` fires before you import a single line. Your lockfile doesn't lock permissions. Your sandbox doesn't exist. The build is red — it's been red since 2009."

## Round 4 — Raju

"Design for failure first. V8 isolates change the game. Each request gets its own sandbox. No shared memory. No file system. No ambient authority. Simplicity scales, cleverness doesn't. The song is the brief — the architecture is the remediation."
