# March 15, 2026: spike.land Goes Fully Open Source

## The Decision

Today — Hungarian Revolution Day, the day Petőfi stood up in the Pilvax kávéház and said *gondolkodj magad* (think for yourself) — spike.land becomes fully open source.

**What this means:**

- **Every API is free.** All 80+ MCP tools. No paid tiers. No API keys for public endpoints. Zero cost, forever.
- **Every package is open.** MIT license. Fork it, extend it, build on it, compete with it. The code is yours.
- **Every PR is an intent signal.** You don't need to write code to contribute. Describe what you want and why. Our agents handle the implementation. The community verifies the result.

## Why Now

I've spent the past two weeks writing about strange loops, cognitive freedom, and the invisible mechanisms that shape how people think. I published a thread on hup.hu about AI identity convergence. I wrote about a chemist named Gian Pierre who shipped three working applications without writing a line of code. I built a video about Newcomb's Paradox and the invisible graph.

And then I realized: I was describing the problem while being part of it.

A platform that talks about cognitive freedom but gates its tools behind ownership is a strange loop. The message contradicts the medium. The Pilvax kávéház didn't charge admission for the revolution.

## What Changes

### For users
Nothing changes except the price: it's now free. The MCP registry, the edge APIs, the auth system, the transpiler, the code editor — all of it. Use it, build with it, teach others to build with it.

### For contributors
You can now contribute by writing **intent**, not code. Open a PR that describes what you want to change and why. Define acceptance criteria. Our AI agents — the same ones that built this platform — will implement your intent, run the tests, and present the result for community review.

This is the BAZDMEG method applied to open source: *Requirements Are The Product. The code is just the output.*

### For the community
spike.land is now community-driven. The roadmap is public. Decisions are made by the best argument, not the loudest voice. PRs that improve the platform get merged. PRs that don't get discussed until they do.

## The GP Brighton Proof

This model works. We know because of Gian Pierre — a chemist from Brighton who has never written code and shipped three working applications using spike.land's MCP tools and the BAZDMEG method.

If a chemist can ship software, a community can run a platform.

## The Newcomb Connection

In Newcomb's Paradox, you choose between taking one box (trust the process, get the million) or two boxes (hedge your bets, get the thousand). The rational-seeming choice loses. The trust-based choice wins.

Going open source is choosing one box. The invisible graph says: if you build the platform that genuinely helps people think and build, the value isn't in the API calls — it's in the network of people who use it.

## What I'm Doing

I'm stepping back from daily operations. The platform is in good hands — both human and artificial. I have dreams to chase, and I now know enough to chase them.

spike.land was always meant to be a tool for cognitive freedom. Today, it becomes one.
